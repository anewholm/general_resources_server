database:class-xschema([string className|node class:*, xschema-name, stand-alone, compiled]). generate the inherited tech version of the xschema for the given Class

caller frees result. generates the amalgamated inherited xschema deifinition for the main class indicated by the parameter. no parameters: the context node will be used and treated as a node (class:*) definition

Parameters:
pQE
pXCtxt
[out]pNodes
@classNamethis string will be used to lookup the main class OR (class:*) this node will be used to evaluate the target main class
@xschema-nameoptional name of the xschema name to return
@stand-aloneall dependent types included (inheritance and element types)
@compiledall inherited fields compiled in to one linear xsd:complexType
See also:
system_transforms/class_xschema.xsl
Returns:
the generated xschema stylesheet
